# To-Do List Life Dashboard 🚀

A simple, clean personal productivity dashboard built with HTML, CSS, and Vanilla JavaScript.

## ✅ Features

### MVP
- 🕐 **Greeting** — Live clock, date, and time-based greeting (Morning/Afternoon/Evening/Night)
- ⏱ **Focus Timer** — Countdown timer with Start, Stop, Reset
- ✅ **To-Do List** — Add, Edit, Mark done, Delete tasks (saved to LocalStorage)
- 🔗 **Quick Links** — Add/remove favorite website shortcuts (saved to LocalStorage)

### Challenges Completed (3/5)
- 🌙 **Light / Dark mode** toggle
- 👤 **Custom name** in greeting
- 🚫 **Prevent duplicate tasks** (case-insensitive check with warning)

- HTML5

- Vanilla JavaScript (no frameworks)
- Browser LocalStorage API

## 🚀 How to Run

Just open `index.html` in any modern browser. No setup required!

Or visit the live site: [GitHub Pages Link]

## 📦 How to Deploy to GitHub Pages

1. Push all files to your GitHub repository
2. Go to **Settings → Pages**
3. Set Source to `main` branch → `/ (root)`
4. Click Save — your site will be live at `https://[username].github.io/[repo-name]`
